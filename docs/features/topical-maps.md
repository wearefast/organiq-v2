# Feature: Topical Maps

## Overview

Topical maps organize keywords into a hierarchical structure of pillars, clusters, and individual keywords. They are generated by the `topical-map` agent from consolidated keyword data and serve as the content planning backbone.

## Key Files

| File | Purpose |
|------|---------|
| `server/src/features/topical-maps/topical-maps.controller.ts` | REST API under `projects/:projectId/topical-maps` |
| `server/src/features/topical-maps/topical-maps.service.ts` | CRUD + stats |
| `server/src/features/topical-maps/topical-maps.module.ts` | NestJS module |
| `frontend/src/app/(dashboard)/workspaces/[wId]/projects/[pId]/topical-map/page.tsx` | Topical map viewer |
| `server/src/agents/definitions/topical-map.agent.md` | Agent that generates the map |

## API Endpoints

| Method | Path | Description |
|--------|------|-------------|
| `GET` | `/projects/:projectId/topical-maps` | List all maps for a project |
| `GET` | `/projects/:projectId/topical-maps/stats` | Aggregate stats |
| `GET` | `/projects/:projectId/topical-maps/:id` | Get single map |
| `POST` | `/projects/:projectId/topical-maps` | Create a map |
| `PATCH` | `/projects/:projectId/topical-maps/:id` | Update a map |
| `DELETE` | `/projects/:projectId/topical-maps/:id` | Delete a map |

## Data Model

### topical_maps table

| Column | Type | Description |
|--------|------|-------------|
| id | UUID | Primary key |
| projectId | UUID | FK → projects |
| workflowRunId | UUID | FK → workflow_runs (nullable) |
| name | text | Map title |
| pillars | jsonb | Hierarchical structure (pillars → clusters → keywords) |
| calendar | jsonb | Optional content calendar derived from the map |

## Structure Format

The `pillars` JSONB contains a hierarchical array:

```json
{
  "pillars": [
    {
      "name": "Running Shoes",
      "description": "Core pillar covering all running shoe topics",
      "clusters": [
        {
          "name": "Best Running Shoes",
          "intent": "commercial",
          "keywords": [
            { "keyword": "best running shoes 2026", "volume": 12000, "difficulty": 45 },
            { "keyword": "top rated running shoes", "volume": 8500, "difficulty": 38 }
          ]
        }
      ]
    }
  ]
}
```
