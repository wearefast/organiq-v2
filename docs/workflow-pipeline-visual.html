Pulse OS — SEO Intelligence Pipeline

18-step AI-powered workflow: from business profile to published content

🎯 What Is This?

Think of this as an assembly line for SEO strategy. Raw materials (your website URL and business info) enter at one end. At each station, a specialized AI worker does one job — researching competitors, analyzing keywords, building a content plan, writing articles, generating images. At the end, you have a complete SEO strategy with ready-to-publish content.

How It Works

🤖 AI Agents

Each step is run by a specialized AI (GPT-4o) with a specific job description and a set of tools it's allowed to use — like a specialized employee.

🔧 Tools = Real Data

Agents don't guess — they call real SEO tools (Ahrefs, Google PageSpeed, Serper, etc.) to get actual data: rankings, traffic, keyword volumes, competitor pages.

✅ Approval Gates

11 of 18 steps pause for your review before continuing. You approve, edit, or reject the output. Nothing proceeds without your sign-off on critical decisions.

🔗 Chain of Context

Every step's output becomes input for the next step. Step 17 (article writing) knows everything from steps 1–16. Context accumulates like a snowball.

📦 How Data Flows Between Steps

1. Agent Runs

AI calls tools (Ahrefs, Serper, etc.), gets real data, then synthesizes it into a structured JSON output following its schema.

2. Output Stored

The JSON output is saved as an "artifact" in the database with version tracking. Each re-run creates a new version.

3. Context Built

When the next step starts, ALL prior step outputs are loaded into a context map: {"business-profile": {...}, "seed-keywords": {...}, ...}

4. Prompt Injected

The context is injected into the next agent's prompt using template variables like {{consolidated-keywords.keywords}}. The AI sees everything.

🔒 Approval Gates: When a step requires approval, the pipeline pauses. You see the output, can edit it, and either approve (which triggers downstream steps) or reject (which stops the chain). 11 of 18 steps have this safety gate. The 7 auto-approved steps are intermediate analysis that feeds into the next approval point.
🧹 Brand Filter: Before the Topical Map step, a special transformation automatically removes competitor brand keywords from the consolidated list. You won't accidentally create content targeting "Competitor Inc reviews" — only your own brand keywords pass through.
✅ Output Validation: Every step's output is checked against its schema before being saved. If the AI returns the wrong format (e.g., a number where an object is expected), the step fails immediately rather than passing bad data downstream.
