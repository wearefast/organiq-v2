# Site Audit Scoring Rubric

## Overall Score: 0-100

### Technical Health (30% weight)
| Score Range | Criteria |
|-------------|----------|
| 90-100 | No critical issues, all CWV green, fully mobile-optimized, clean crawl |
| 70-89 | Minor issues, mostly green CWV, mobile-friendly, few crawl warnings |
| 50-69 | Some blocking issues, mixed CWV, partial mobile optimization |
| 30-49 | Multiple blocking issues, poor CWV, significant mobile problems |
| 0-29 | Critical failures, site barely crawlable or indexable |

### On-Page SEO (25% weight)
| Score Range | Criteria |
|-------------|----------|
| 90-100 | All pages optimized, unique titles/metas, strong heading structure |
| 70-89 | Most pages optimized, minor duplication, good heading usage |
| 50-69 | Mixed optimization, some missing metas, inconsistent headings |
| 30-49 | Poor optimization, many missing elements, weak structure |
| 0-29 | Minimal optimization, bulk missing metadata |

### Content Quality (25% weight)
| Score Range | Criteria |
|-------------|----------|
| 90-100 | All content unique, fresh, comprehensive, well-structured |
| 70-89 | Mostly unique, recently updated, good depth |
| 50-69 | Some thin/duplicate content, moderate freshness |
| 30-49 | Significant thin content, stale pages, duplication issues |
| 0-29 | Mostly thin/duplicate, severely outdated |

### Schema & Structure (20% weight)
| Score Range | Criteria |
|-------------|----------|
| 90-100 | Comprehensive schema markup, clean URL structure, strong internal linking |
| 70-89 | Good schema coverage, logical URLs, decent internal linking |
| 50-69 | Partial schema, acceptable URLs, some linking gaps |
| 30-49 | Minimal schema, messy URLs, poor internal linking |
| 0-29 | No schema, broken URL patterns, no internal linking strategy |
