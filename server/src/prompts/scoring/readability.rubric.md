# Readability Scoring Rubric

## Score: 0-100

| Score | Grade | Criteria |
|-------|-------|----------|
| 90-100 | A+ | Flesch-Kincaid grade 6-8, short sentences, simple vocabulary, scannable |
| 80-89 | A | Grade 8-10, mostly short sentences, accessible vocabulary |
| 70-79 | B | Grade 10-12, moderate sentence length, some technical terms explained |
| 60-69 | C | Grade 12-14, longer sentences, technical vocabulary present |
| 50-59 | D | Grade 14+, complex sentences, dense technical language |
| 0-49 | F | Academic-level complexity, inaccessible to general audience |

## Factors
- Average sentence length (target: 15-20 words)
- Paragraph length (target: 2-4 sentences)
- Passive voice usage (target: <10%)
- Transition word usage (target: >30% of sentences)
- Subheading distribution (target: every 200-300 words)
