# SEO Quality Scoring Rubric

## Score: 0-100

| Score | Grade | Criteria |
|-------|-------|----------|
| 90-100 | A+ | All on-page elements optimized, strong keyword integration, perfect structure |
| 80-89 | A | Most elements optimized, natural keyword usage, good structure |
| 70-79 | B | Key elements present, adequate keyword usage, acceptable structure |
| 60-69 | C | Some elements missing, keyword stuffing or under-optimization |
| 50-59 | D | Multiple elements missing, poor keyword integration |
| 0-49 | F | Minimal optimization, no clear keyword targeting |

## Factors
- Title tag optimization (keyword position, length 50-60 chars)
- Meta description (compelling, keyword-rich, 150-160 chars)
- H1 tag (single, keyword-rich, unique)
- H2/H3 structure (logical hierarchy, keyword variations)
- Keyword density (target: 1-2% primary, 0.5-1% secondary)
- Internal linking (3-5 relevant internal links)
- Image optimization (alt tags, file names, compression)
- URL structure (short, keyword-inclusive, hyphenated)
- Schema markup (appropriate type implemented)
