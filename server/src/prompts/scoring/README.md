# Scoring Rubrics Index

This directory contains scoring rubrics used by agents to evaluate outputs.

## Available Rubrics

- `readability.rubric.md` — Flesch-Kincaid, sentence complexity, vocabulary level
- `seo-quality.rubric.md` — On-page optimization scoring
- `citability.rubric.md` — AI citation readiness scoring
- `opportunity.rubric.md` — Keyword opportunity scoring formula
- `content-length.rubric.md` — Content depth adequacy scoring
