# AI Citability Scoring Rubric

## Score: 0-100

| Score | Grade | Criteria |
|-------|-------|----------|
| 90-100 | A+ | Highly citable — clear definitions, structured data, authoritative claims, Q&A format |
| 80-89 | A | Well-structured for AI extraction, strong factual statements |
| 70-79 | B | Decent structure, some extractable facts, could be more direct |
| 60-69 | C | Moderate citability, buried key information, verbose |
| 50-59 | D | Poor citability, no clear definitions, unstructured prose |
| 0-49 | F | Not citable — opinion-heavy, no factual statements, no structure |

## Factors
- Direct answer patterns (question → concise answer in first sentence)
- Definition patterns ("X is Y" statements)
- List and table usage (easily extractable)
- Factual claim density (statistics, data points, named entities)
- Source attribution (E-E-A-T signals)
- Schema markup (FAQ, HowTo, Article structured data)
- Content freshness signals (dates, "updated" markers)
- Expertise signals (author credentials, citations)
