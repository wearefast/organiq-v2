# Report Templates

Templates for PDF report generation via the Python sidecar.

## Available Templates (to be implemented in Phase H)

- `full-strategy.template.md` — Complete strategy report (all 17 steps)
- `ai-visibility.template.md` — AI/GEO visibility focused report
- `keyword-research.template.md` — Keyword research summary report
- `content-plan.template.md` — Content plan and topical map report
